# 🌐 Personal Portfolio Website

This is my personal portfolio website built using HTML, CSS, and JavaScript.  
The website showcases my skills, projects, education, and contact details with a modern and responsive design.

## 🚀 Features

- Responsive Portfolio Design
- About Me Section
- Skills Section
- Projects Showcase
- Contact Section
- Smooth Navigation
- Clean UI Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

- Rock Paper Scissors Game
